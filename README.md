Project Title: AUTONOVA

Developed by:
Sam Lloyd Lumba,
Samantha Lauren Balundo,
Ken Macaraeg

AUTONOVA is a modern and interactive web platform designed for car enthusiasts. It serves as a digital space where users can explore various car models, learn about their features, and engage with automotive-related content. Built using HTML and CSS, the project emphasizes responsive design, user-friendly navigation, and clean visual structure. AUTONOVA showcases how web technologies can be used to create an engaging, informative, and visually appealing experience for users interested in automobiles.
